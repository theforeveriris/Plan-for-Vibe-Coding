
## 三、快速获取流程

### MCP Server 获取步骤

```bash
# 1. 浏览 awesome-mcp-servers 列表
# 访问： `https://github.com/punkpeye/awesome-mcp-servers` 

# 2. 找到需要的 Server（如 filesystem）
# 查看其 GitHub 仓库和安装说明

# 3. 安装（通常通过 npx 或 pip）
npx -y @modelcontextprotocol/server-filesystem /path/to/dir

# 4. 配置到 AI IDE（Cursor/Trea/Windsurf）
# 编辑对应的 mcp.json 配置文件
```

### Skills 获取步骤

```bash
# 方式一：通过 Claude Code 自然语言安装
# 在 Claude Code 中输入：
帮我安装 skill，项目地址是： `https://github.com/anthropics/skills/tree/main/skills/pptx` 

# 方式二：手动安装
# 1. 从官方仓库或社区找到 Skill
# 2. 下载包含 SKILL.md 的文件夹
# 3. 复制到 ~/.claude/skills/（全局）或项目 .claude/skills/（项目级）

# 方式三：使用 find-skills 搜索
# 在 Claude Code 中输入：
使用 find-skills 搜索 react testing
```
