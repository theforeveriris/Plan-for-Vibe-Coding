# Session 2026-04-24-006

## 任务
修复网络连接错误、SSE 重连失败和 Hashrate 为 0 的问题

## 问题分析

### 1. "Failed to execute testFn: TypeError: Failed to fetch"
- **原因**：前端默认使用 `http://localhost:3000`，但后端实际运行在 `http://localhost:3002`
- **影响**：所有 API 请求都失败，前端无法与后端通信

### 2. SSE 连接频繁断开并重连（已达10次上限）
- **原因1**：`EventSource.onerror` 在连接建立前就会触发，代码没有区分初始连接错误和连接中断
- **原因2**：前端 `onopen` 中没有重置重连计数器
- **影响**：重连计数器迅速累加到上限，SSE 连接完全失败

### 3. Hashrate 持续为 0.0 keys/s
- **原因1**：使用 RSA 2048 生成密钥，每个密钥需要 200-500ms
- **原因2**：Worker 线程在 Next.js 开发模式下无法解析 ESM 模块
- **影响**：实际没有密钥生成，hashrate 始终为 0
