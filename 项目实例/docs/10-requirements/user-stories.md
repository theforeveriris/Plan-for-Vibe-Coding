# 用户故事（User Stories）

## US-001: 作为用户，我想启动挖掘任务挖掘包含特殊模式的PGP密钥
**优先级**：P0 | **故事点**：8 | **迭代**：Sprint 1

### 验收标准（AC）
- [ ] AC1: 用户可选择规则类型（连续数字/回文/自定义正则等）
- [ ] AC2: 用户可设置规则参数（如连续数字的最少位数）
- [ ] AC3: 用户可设置并行线程数（1-8）
- [ ] AC4: 点击"开始"后任务启动，进度开始实时更新
- [ ] AC5: 点击"停止"后任务立即停止

### 技术任务（拆分为 Task）
- [ ] T001: 实现 PGP 密钥生成核心逻辑
- [ ] T002: 实现多种筛选规则
- [ ] T003: 实现 SSE 实时推送

## US-002: 作为用户，我想实时查看挖掘进度和统计数据
**优先级**：P0 | **故事点**：5 | **迭代**：Sprint 1

### 验收标准（AC）
- [ ] AC1: 实时显示已尝试密钥总数
- [ ] AC2: 实时显示当前hashrate（keys/秒）
- [ ] AC3: hashrate变化以折线图展示
- [ ] AC4: 挖掘日志实时滚动显示

### 技术任务（拆分为 Task）
- [ ] T004: 前端统计面板开发
- [ ] T005: hashrate 图表集成

## US-003: 作为用户，我想查看和管理已发现的特殊密钥
**优先级**：P0 | **故事点**：5 | **迭代**：Sprint 2

### 验收标准（AC）
- [ ] AC1: 已发现密钥以卡片形式展示
- [ ] AC2: 卡片显示：指纹（高亮匹配部分）、规则标签、生成时间、难度评级
- [ ] AC3: 支持一键复制公钥
- [ ] AC4: 支持下载.asc文件
- [ ] AC5: 支持分页/无限滚动加载

### 技术任务（拆分为 Task）
- [ ] T006: 密钥卡片组件开发
- [ ] T007: SQLite 存储和查询
- [ ] T008: 导出功能实现

## US-004: 作为高级用户，我想使用自定义正则表达式筛选密钥
**优先级**：P1 | **故事点**：3 | **迭代**：Sprint 2

### 验收标准（AC）
- [ ] AC1: 用户可输入自定义正则表达式
- [ ] AC2: 前端提供正则语法校验
- [ ] AC3: 正则错误时显示友好提示

### 技术任务（拆分为 Task）
- [ ] T009: 自定义正则筛选器

## US-005: 作为用户，我想暂停和恢复挖掘任务
**优先级**：P1 | **故事点**：5 | **迭代**：Sprint 2

### 验收标准（AC）
- [ ] AC1: 暂停后任务状态保留
- [ ] AC2: 恢复后继续之前的进度
- [ ] AC3: 暂停期间不消耗服务器资源

### 技术任务（拆分为 Task）
- [ ] T010: 任务暂停/恢复功能

## US-006: 作为用户，我想看最难发现的密钥排行榜
**优先级**：P2 | **故事点**：3 | **迭代**：Sprint 3

### 验收标准（AC）
- [ ] AC1: 显示尝试次数最多的top 10密钥
- [ ] AC2: 排行榜实时更新
- [ ] AC3: 可按规则类型筛选

### 技术任务（拆分为 Task）
- [ ] T011: 排行榜功能开发
