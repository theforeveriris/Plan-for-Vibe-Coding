# 性能指标（Performance SLA）

## 挖掘性能

| 指标 | 目标值 | 测量方式 |
|------|--------|---------|
| 单线程密钥生成速度 | 10-15 keys/sec | 实验室环境 |
| 4线程密钥生成速度 | 30-50 keys/sec | 实验室环境 |
| 最大并行任务数 | 5 | 系统限制 |
| 最大线程数/任务 | 8 | 防止资源耗尽 |

## API性能

| 指标 | 目标值 | 备注 |
|------|--------|------|
| SSE推送延迟 | < 100ms | 从生成到推送 |
| API响应时间 | < 200ms | POST /api/miner/start |
| 密钥列表查询 | < 500ms | GET /api/keys (100条) |

## 前端性能

| 指标 | 目标值 | 备注 |
|------|--------|------|
| 首屏加载时间 | < 2s | Lighthouse |
| SSE重连时间 | < 1s | 网络断开后 |
| 内存占用 | < 200MB | 持续运行1小时 |

## 稳定性

| 指标 | 目标值 | 备注 |
|------|--------|------|
| 任务意外中断率 | < 1% | 服务器问题导致 |
| 数据持久化成功率 | 100% | 关键数据不丢失 |
