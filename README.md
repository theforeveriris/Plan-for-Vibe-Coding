# Vibe Coding 技巧

个人自用的 Vibe Coding 技巧和方法仓库。

## 简介

本仓库整理了 Vibe Coding（AI 辅助编程）的学习资料、环境配置教程、工作流方法和工具使用技巧。

## 演示工具

本仓库中的演示主要使用以下工具：
- **Claude Code**：AI 代码助手
- **Trae CN**：主要演示工具

## 图片资源说明

本仓库中的所有图片资源均来自于：
- **Mermaid 绘制**：用于流程图、架构图等图表
- **演示工具使用截图**：从实际工具使用中截取的示例
