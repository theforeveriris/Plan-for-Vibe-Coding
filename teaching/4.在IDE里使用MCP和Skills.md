# 在 IDE 里使用 MCP 和 Skills

本教程详细介绍如何在 Trea IDE 中配置和使用 MCP 协议以及 Skills。

## 一、Trea 中的 MCP 配置

Trea 是国内字节跳动推出的 AI 编程 IDE，原生支持 MCP 协议。

### 1. 环境准备

**基础要求**：

- Node.js (NJS)：用于运行 MCP Server
- UV/UVX：Python MCP Server 所需
- Python 3.10+：作为 UVX 的前提

```bash
# 验证 Node.js
node --version
npm --version

# 安装 uv（如果需要 Python MCP Server）
pip install uvx
```

### 2. 通过 MCP 市场安装（推荐）

**步骤**：

1. 更新 Trea 到最新版本
2. 点击右上角 "AI 侧栏" → 设置图标 → MCP
3. 在 MCP 市场中选择需要的服务（如 Apifox、DeepSeek 等）
4. 填入必要的 API Key 或配置信息
5. 保存配置

### 3. 手动配置 MCP Server

**示例：添加 Apifox MCP Server**：

```json
// macOS/Linux 配置
{
  "mcpServers": {
    "DeepSeek API - API 文档": {
      "command": "npx",
      "args": [
        "-y",
        "apifox-mcp-server@latest",
        "--site-id=5435161"
      ]
    }
  }
}
```

```json
// Windows 配置（如果上面的不工作）
{
  "mcpServers": {
    "DeepSeek API - API 文档": {
      "command": "cmd",
      "args": [
        "/c",
        "npx",
        "-y",
        "apifox-mcp-server@latest",
        "--site-id=5435161"
      ]
    }
  }
}
```

### 4. 使用 MCP

**配置完成后**：

1. 在 Trea 聊天框中点击 "智能体"
2. 选择 "Builder with MCP"
3. 直接通过自然语言使用，例如：
   - "查询这个 API 的文档"
   - "根据接口文档生成代码"

## 二、Trea 中的 Skills 使用

Trea 本身不直接支持 Skills 文件，但可以通过 skillport-mcp 桥接使用。

## 相关资源

- **[常见的 MCP 和 Skills 获取平台](../extended%20docs/4.a%20常见的mcp和skills获取平台.md)**：详细介绍如何获取和安装各种 MCP Server 和 Skills

### 1. 安装 skillport

```bash
# 确保已安装 Python3
pip3 install uvx

# 安装 skillport
pip3 install skillport
# 或：uv tool install skillport
```

### 2. 配置 skillport-mcp

在 Trea 的 MCP 配置中添加：

```json
{
  "mcpServers": {
    "skillport": {
      "command": "uvx",
      "args": ["skillport-mcp"],
      "env": {
        "SKILLPORT_SKILLS_DIR": "~/.skillport/skills"
      }
    }
  }
}
```

### 3. 使用 Skill

1. 将 Skill 文件（SKILL.md）放入 \~/.skillport/skills/ 目录
2. 在 Trea 中选择 "Builder with MCP"（确保包含 skillport）
3. 提问时加上前缀：基于 skill 解答，\[你的问题]

**示例**：

```
基于 skill 解答，电脑性能优化
```
